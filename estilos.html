Estilos usando 'Style'

Aqui vamos a usar las distintias funciones de style para dar formato.

Su estructura es: < etiqueta style="propiedad:valor;" >

Para cambniarle el color al fondo de la pagina se usa style="background-color:color";

Para ponerle un fondo distinto a cada cosa se utiliza lo mismo pero con su respectiva etqiueta al principio.
